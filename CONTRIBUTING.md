## How to Contribute
1. **Fork the Repository:** Create your own branch from `main`.
2. **Standard Naming:** We follow strict Rust conventions (UpperCamelCase for Structs/Enums, snake_case for functions).
3. **Tests:** Ensure all unit and integration tests pass.
   ```bash
   cargo test --workspace
   ```
4. Pull Requests: Provide a clear description of your changes and link any related issues.

## Development Setup

1. Build: `cargo build`
2. Lint: `cargo clippy`
3. Format: `cargo fmt`
